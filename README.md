# Show/Hide Each Element Panel Plugin for QGIS

QGIS plugin that adds a panel to show/hide individual features in vector layers.

## Installation

1. Copy the plugin folder to your QGIS plugins directory
2. Restart QGIS or reload plugins
3. Enable the plugin in Plugins → Manage and Install Plugins

## Usage

1. Click the plugin toolbar button to open the panel
2. Select a layer from the list
3. Check/uncheck features to show/hide them on the map
4. Use the ⚙ button for settings (attribute columns, filters, feature limits)

## Requirements

- QGIS 3.40 or higher
